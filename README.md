# TerraTwin Garden Map

A simple Cloudflare Pages project that lets someone enter a city, estimates a USDA-style hardiness zone from recent historical lows, highlights similar climate regions on a flat global map, and recommends outdoor plants from a 1,000+ record plant database.

## Run locally

```bash
npm install
npm run dev
```

## Build for Cloudflare Pages

```bash
npm run build
```

Cloudflare Pages settings:

- Build command: `npm run build`
- Build output directory: `dist`
- Node version: 24 or current LTS

## Data model

- City search uses the Open-Meteo Geocoding API.
- USDA-zone estimation uses Open-Meteo historical daily minimum temperature data.
- Regional climate fit uses NASA POWER climatology for monthly `T2M_MAX`, `T2M_MIN`, and `RH2M`.
- Map analog regions are re-ranked after city search using cached NASA POWER normals for warmest high, coolest normal low, and mean relative humidity.
- USDA-style zones are calculated from the average of recent annual extreme lows.
- Plant data is generated by `scripts/generate-plants.mjs` into `public/plants.json`, including category, plant type, USDA zone range, preferred warm-season high range, and relative-humidity range.

USDA hardiness zones are only a winter cold-tolerance signal. The UI calls out that soil, heat, humidity, rainfall, pests, and microclimates still matter.
